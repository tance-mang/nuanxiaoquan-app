\# 暖小圈

\### 基于状态感知的智能学习行为调节系统｜State-Aware Adaptive Learning System



\*\*一款真正懂你状态的学习闭环引擎\*\*  

不做打卡、不做计划、不做社区——\*\*只做学习行为的智能调节\*\*  

让你\*\*无需思考，一键开始，持续专注\*\*



\---



\## 核心定位

暖小圈是\*\*完全自研\*\*的\*\*学习行为闭环系统\*\*，通过实时感知用户精力、能量与行为数据，自动计算当日最优学习强度，动态输出轻启动/标准/冲刺三种专注模式，从根源降低学习启动门槛，实现可持续的深度学习。



\*\*不是普通学习 App，是一套可解释、可复现、可闭环的学习行为算法系统。\*\*



\---



\## 核心亮点

\- \*\*自研强度计算引擎\*\*  

&#x20; `score = 状态系数 × 能量系数 × 行为系数`，全本地确定性算法，可单元测试、可追溯、断网可用

\- \*\*完整行为闭环\*\*  

&#x20; 状态录入 → 强度计算 → 模式推荐 → 专注执行 → 中断记录 → 学习反馈 → 次日自动优化

\- \*\*AI 架构分层设计\*\*  

&#x20; 外部 AI 仅负责文案生成，\*\*核心判定/分析/预测全由本地算法实现\*\*，无黑盒、无隐私风险

\- \*\*女性关怀性别门控\*\*  

&#x20; 经期功能严格按性别权限开放，合规、隐私、人性化

\- \*\*跨平台零冗余\*\*  

&#x20; Flutter 纯跨平台架构，0 Web-only 代码，支持 Android / iOS / Web 三端一致运行

\- \*\*行为感知与主动干预\*\*  

&#x20; 微休息提醒、后台回归提示、专注里程碑，全程无打扰式关怀

\- \*\*AI 小暖悬浮交互\*\*  

&#x20; 3D 渲染 + Lissajous 动态漂浮 + 呼吸动画，全页面全局同步



\---



\## 技术栈

\- \*\*前端\*\*：Flutter 3.29.x + Dart 3.3.x + GetX

\- \*\*存储\*\*：shared\_preferences 本地持久化

\- \*\*AI 层\*\*：豆包 API（文案生成）+ 本地 Python 算法（分析/预测）

\- \*\*架构\*\*：服务层分离、低耦合、可测试、可闭环

\- \*\*兼容\*\*：Android 6.0+ / iOS 11.0+



\---



\## 核心功能

1\. \*\*每日状态感知\*\*：精力滑块 + 状态标签 → 自动推断能量等级

2\. \*\*智能强度引擎\*\*：3 种系数 → 5 种输出模式 → 精准推荐时长

3\. \*\*Hero CTA 一键开始\*\*：支持三级引导风格 + 手动微调 ±5 分钟

4\. \*\*沉浸式自习室\*\*：暖圈钟 + 扩散涟漪 + 夕阳进度 + 动态成员效果

5\. \*\*学习反馈闭环\*\*：结束三档反馈 → 写入行为日志 → 优化次日推荐

6\. \*\*知识小馆智能推荐\*\*：按学习模式匹配治愈/励志/放松内容

7\. \*\*AI 小暖助手\*\*：学习计划生成、对话交互、历史会话管理

8\. \*\*暖圈关怀（女性专属）\*\*：周期记录、阶段预测、状态适配建议
